## SECTION 8: REFERENCE ART STYLES

### Claymation Style Reference
When prompting for scenes featuring our characters, use language like:
- "Claymation CGI rendering"
- "Stop-motion character meets digital illustration"
- "Hand-sculpted material aesthetic with digital polish"
- "Coarse texture meets digital precision"

### Philosophical Aesthetic Language
Use these terms consistently:
- "Contemplative" (for introspective scenes)
- "Ethereal" (for mystical themes)
- "Monumental" (for classical themes)
- "Intricate" (for pattern/connection scenes)
- "Cosmic" (for universal themes)

### Color Consistency
Always reference the specific color codes when possible:
- Deep Teal: #0D3D47
- Gold: #D4AF37
- Warm White: #F5F1E8
- Wisdom Bronze: #8B7355
- Insight Purple: #6A3E86

## HOW TO USE THESE PROMPTS

### For Midjourney:
1. Copy the prompt exactly
2. Type `/imagine` in Discord
3. Paste the prompt
4. Adjust parameters: `--ar 16:9` for aspect ratio, `--quality 2` for detail
5. Allow 1-2 minutes for generation

### For DALL-E 3 (ChatGPT Plus):
1. Paste prompt into ChatGPT
2. Ask: "Generate this image with DALL-E 3"
3. Refine with feedback: "Adjust the lighting" or "Add more golden tones"
4. Download high-resolution version

### For Stable Diffusion (Local/Replicate):
1. Use the prompt exactly or shorten for token limits
2. Set seed for consistency across batches
3. Adjust sampler steps (50+ recommended for quality)
4. Use negative prompts to avoid unwanted elements

## NOTES FOR BEST RESULTS

1. **Consistency is key**: Save successful prompts and regenerate with minor variations
2. **Test multiple generators**: Midjourney excels at aesthetic, DALL-E 3 at specific details
3. **Color anchoring**: Always include specific hex codes or color references
4. **Character consistency**: Use character description details consistently
5. **Resolution matters**: Always specify aspect ratio and resolution for web/NFT use
6. **Backup originals**: Keep all generated images in `/art/placeholder-generations/`
7. **Quality over quantity**: It's better to have 10 amazing images than 100 mediocre ones
