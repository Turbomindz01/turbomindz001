## STEP 1: MetaMask Setup for Polygon (15 minutes)

### If MetaMask isn't installed:

1. Go to metamask.io → Download → Add to Chrome
2. Create new wallet → write down seed phrase on PAPER (never digital)
3. Set a strong password

### Add Polygon Network:

1. Click the network dropdown (top of MetaMask, says "Ethereum Mainnet")
2. Click "Add Network" → "Add a network manually"
3. Enter these EXACT values:
   - Network Name: Polygon Mainnet
   - New RPC URL: https://polygon-rpc.com
   - Chain ID: 137
   - Currency Symbol: MATIC
   - Block Explorer URL: https://polygonscan.com
4. Click Save → Switch to Polygon Mainnet

### Get MATIC:

- **Option A:** Buy on Coinbase/Binance → Send to your MetaMask Polygon address
- **Option B:** Use a faucet for tiny amounts (Google "Polygon MATIC faucet")
- You need very little — transactions cost <$0.01 on Polygon

## TROUBLESHOOTING:

- **MetaMask won't connect to OpenSea:** Try refreshing, clearing cache, or using a different browser
- **Transaction stuck:** Polygon gas is usually instant; if stuck, check polygonscan.com for your transaction
- **Image upload fails:** Make sure images are under 50MB, PNG or JPG format
- **Can't find Polygon option:** Make sure you added the Polygon network to MetaMask first
