## How to Use These Documents

### If You're Starting From Scratch:
1. Read **MASTER_BIBLE_V2.md** — understand the soul of the project
2. Read **STORY_BIBLE.md** — understand the narrative
3. Read **PRODUCTION_PIPELINE.md** — understand the daily workflow
4. Read **DEVELOPMENT_GUIDE.md** — set up your dev environment
5. Start building

### Daily Operation:
1. Say **"Today's Tasks"** to Claude
2. Follow the **DAILY_COMMAND_TEMPLATE.md** output
3. Generate images → Generate video → List NFTs → Post content
4. Every 7 days: assemble episode + send newsletter

### Key Principles:
- **4 NFTs per day.** Non-negotiable.
- **Every NFT is a film frame.** Double duty.
- **Dark mode default.** Deep teal + gold + warm white.
- **Philosophy first.** Questions, not products.
- **Build in public.** Share everything.
