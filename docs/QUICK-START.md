# TURBOMINDZ — QUICK START GUIDE
## How to Use This Workspace in Claude Desktop Cowork Mode

---

## Step 1: Open Cowork Mode
1. Open Claude Desktop on Windows
2. Switch to Cowork mode (bottom-left toggle or sidebar icon)
3. Select this folder as your workspace (the Turbomindz folder)

## Step 2: Verify Setup
Say to Claude: **"Show me a summary of my workspace"**

Claude should see:
- 10 custom skills in `.skills/skills/`
- Folder structure with products/, content/, art/, app/, docs/, marketing/, financials/
- Template files in docs/ and financials/
- An example NFT scene in art/nft-scenes/

## Step 3: Your First Session

### Generate your first daily briefing:
> "COSMO: Give me today's daily briefing. This is Production Day 1, Sprint Week 1."

### Create your first product:
> "VAULT: Create a set of 5 Stoicism quote cards as PDFs for Gumroad."

### Generate an NFT scene:
> "MINT: Generate the complete scene package for TM-151, Universe 7: Crystalline Dreamscape, Theme: Poetry & Aesthetics."

### Write your first article:
> "NOVA: Write a Medium article about what Stoicism actually means versus Instagram Stoicism."

### Create a social media batch:
> "STELLA: Generate 3 days of social media content for a scene-reveal of TM-151."
