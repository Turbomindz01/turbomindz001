## WHAT THE /week COMMAND PRODUCES

When you type `/week 1` (or any week number), Claude produces ALL of the following for 7 days:

### DAILY PRODUCTION (per day, 7 days = full week):

## PARALLEL SUB-AGENT ARCHITECTURE

The /week command uses parallel processing:

**Batch 1 (launch simultaneously):**
- Sub-agent A: Generate all 21 NFT packages (internally parallelized in groups of 7 per day)
- Sub-agent B: Generate all 14 Instagram posts + MJ prompts
- Sub-agent C: Generate all 14 Twitter posts

**Batch 2 (launch simultaneously after Batch 1):**
- Sub-agent D: Generate all 7 Discord announcements
- Sub-agent E: Generate all 21 Pinterest pins
- Sub-agent F: Generate the Medium article

**Batch 3 (final compilation):**
- Sub-agent G: Generate YouTube script
- Sub-agent H: Compile newsletter draft from week's highlights
- Orchestrator: Compile WEEK_OVERVIEW.md with links to all content

## SCHEDULING PLAN FOR ADVANCE PRODUCTION

**Sunday Power Session (2-3 hours):**
1. Run `/week [N+1]` — generates next week's full content package
2. Review the WEEK_OVERVIEW.md
3. Run all MJ prompts in Midjourney (batch the 21 NFT + 14 Instagram + 21 Pinterest = ~56 images)
4. Schedule Instagram posts using Later/Buffer (free tier: 30 posts/month)
5. Schedule Twitter posts using TweetDeck/Buffer
6. Queue Pinterest pins using Tailwind (free tier: 20 pins/month) or manual
7. Save Discord announcements for daily posting (1 min/day to paste)
8. Medium article ready for publishing mid-week
9. YouTube script ready for recording whenever time allows
10. Newsletter draft ready for Friday send

**This means the founder's daily task during the week is just:**
- 5 min: Post the pre-written Discord announcement
- 5 min: Check engagement, reply to comments
- 5 min: Upload and list the day's 3 NFTs on OpenSea (images already made on Sunday)
- 15-30 min: Any remaining production tasks

**Total weekday time: 15-45 minutes/day** (down from 1-2 hours because content is pre-produced)

## DEPLOYMENT WORKFLOW

### Step 1: Content Generation
- User runs `/week [N]` in Claude
- All 79 pieces generated to /TMDZ/weekly-production/week-[N]/

### Step 2: Image Generation
- Batch all MJ prompts from week's content (~56 total images)
- Run in Midjourney (can be done in parallel)
- Save all images with naming convention: TM-[NUMBER]-[platform].png
- Store in appropriate subfolder

### Step 3: Scheduling
- Instagram: Use Later or Buffer (free tier allows scheduling)
- Twitter: Use TweetDeck or Buffer for scheduling
- Pinterest: Use Tailwind or manual scheduling
- Discord: Copy-paste ready, post daily at consistent time
- Medium: Publish mid-week (Wednesday-Thursday ideal)
- YouTube: Have script ready, record on own schedule
- Newsletter: Finalize Friday morning, send Friday afternoon

### Step 4: Daily Execution (Monday-Sunday)
Each day during production week:
1. Copy Discord announcement from daily folder
2. Paste in #announcements channel
3. Upload day's 3 NFT images to OpenSea
4. Create OpenSea listings (metadata auto-populates from JSON files)
5. Set prices (from pricing tier files)
6. List publicly
7. Monitor engagement, reply to comments
8. Share ethusiasm in Discord
